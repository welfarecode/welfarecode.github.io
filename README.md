# WELFARE-CODE HOMEPAGE BY VUE

## 프로젝트 세팅 (Add node_modules folder)
```
npm install
```

### 개발을 위한 가상 서버 열기(Representation : localhost:8080)
```
npm run serve
```

### 실제 웹사이트 개발을 위한 PRODUCTION 만들기 (VUE FILES -> HTML FILES)
```
npm run build
```

### Lint하기
```
npm run lint
```
